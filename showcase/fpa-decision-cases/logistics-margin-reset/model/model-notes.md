# Logistics Margin Reset - Model Notes

This folder contains a public-safe, synthetic FP&A work sample for portfolio use.

## Files

- `../index.html`: CFO decision narrative, KPIs, EBITDA bridge, cash panel, and QA evidence.
- `../model.html`: model walkthrough and workbook explanation.
- `generated-fpa-model.xlsx`: compact generated Excel model with assumptions, scenario output, EBITDA bridge, 13-week cash forecast, and QA checks.

## Data Policy

All data is synthetically generated for demonstration. Do not replace this workbook with company data unless it has been fully anonymized and audited.

Never publish:

- real customer names
- vendor names
- invoice or shipment IDs
- employee names or emails
- actual branch/entity financials
- local file paths
- PBIX/PBIT files with imported data

## Update Checklist

1. Update workbook assumptions and scenario output.
2. Confirm `QA_Checks` shows OK.
3. Update KPI values in `index.html`.
4. Update model summary values in `model.html`.
5. Run local website check before pushing.

## Public Positioning

Use this case to show FP&A judgment:

- what changed
- why it changed
- financial impact
- business owner
- next action
- model control

The goal is to signal executive-ready decision support, not just dashboard building.
